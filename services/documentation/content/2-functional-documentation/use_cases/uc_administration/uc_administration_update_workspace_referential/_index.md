## Global Concepts

The "Update Platform Referential" feature in the Administration panel allows G4IT workspace administrators to maintain workspace reference data used for calculations without direct database access.

This is crucial for keeping impact factors and hypotheses up-to-date with the latest environmental research or organizational specifics.
